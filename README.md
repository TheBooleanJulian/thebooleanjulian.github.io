# thebooleanjulian.github.io

Personal landing page hosted on GitHub Pages at [thebooleanjulian.github.io](https://thebooleanjulian.github.io).

## What it does

- Fetches all public repos from the GitHub API at page load
- Filters out forks, archived repos, and meta repos
- Renders project cards sorted by most recently pushed, with language colours, topic tags, and star counts
- No build step — plain HTML, CSS, and vanilla JS

## Development

Open `index.html` directly in a browser. No dependencies or build tools required.
